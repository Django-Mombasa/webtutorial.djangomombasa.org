# Classes and IDs

Add a `class` or `id` attribute in HTML, then target it in CSS:

```html
<p class="note">Remember to save your work.</p>
```

```css
.note {
  background-color: #fff3cd;
  padding: 8px;
  border-left: 4px solid #f0ad4e;
}
```

Use **classes** when multiple elements share styles. Use **IDs** only
for one-of-a-kind elements.
