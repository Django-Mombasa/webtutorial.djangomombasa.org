# PyCharm

**PyCharm** is the Python IDE made by JetBrains, and the one we
recommend as your starting point for Django. It understands Python
projects deeply — imports, virtual environments, Django templates,
migrations, and tests all get first-class support.

## Editions

- **PyCharm Community** — free, open source, and enough for most
  early Django work.
- **PyCharm Professional** — paid; adds built-in Django support,
  database tools, and remote interpreters. Students and open-source
  maintainers can request a free licence.

Download from
[jetbrains.com/pycharm](https://www.jetbrains.com/pycharm/).

## What you get

- Smart autocomplete and inline type hints for Python.
- A one-click way to create and activate a virtual environment.
- A visual debugger — set a breakpoint, run, and inspect variables.
- A built-in terminal and Git panel so you never leave the editor.

If you are brand new to programming, installing PyCharm Community and
opening your first `hello.py` inside it is a great way to start.
