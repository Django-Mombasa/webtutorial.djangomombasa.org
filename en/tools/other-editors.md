# Other editors

You do not need to use an IDE. Many Django developers are happy with
a lighter editor plus a few extensions.

## VS Code

[**Visual Studio Code**](https://code.visualstudio.com/) (VS Code) is
a free editor from Microsoft. It is fast, highly customisable, and
has extensions for almost everything:

- The **Python** extension for linting, debugging, and virtual
  environments.
- **Pylance** for type-aware completion.
- **Django** extensions for template syntax highlighting.
- **GitLens** for richer Git integration.

## Cursor

[**Cursor**](https://www.cursor.com/) is a fork of VS Code with AI
features built into the editor — chat, inline edits, and agent-style
refactors that can act on multiple files at once. If you like VS Code
but want AI assistance as a first-class feature, Cursor feels
familiar from day one.

## Picking one

- If you want **a full toolkit out of the box** → PyCharm.
- If you want **a light, extensible editor** → VS Code.
- If you want **AI-first** → Cursor.

Any of these will take you through this tutorial without trouble.
Pick the one that feels right and keep going.
