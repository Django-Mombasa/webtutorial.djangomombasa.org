# What is an IDE?

An **IDE** (Integrated Development Environment) is one program that
bundles together everything you need to write software:

- A code editor with syntax highlighting and autocomplete.
- A project view and file explorer.
- A built-in terminal for running commands.
- Debugging, testing, and version-control (Git) integrations.
- Tools that understand your language — refactoring, navigation,
  and error checking.

A plain **editor** does much of the same work but relies more on
external tools and extensions. For Django, either approach works; a
full IDE just gives you more out of the box.
