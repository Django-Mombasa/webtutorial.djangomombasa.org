# Working with GitHub

1. Create an account at [github.com](https://github.com/).
2. Create a new repository on GitHub — do **not** initialise it with a
   README if you already have one locally.
3. Connect your local project and push:

   ```bash
   git remote add origin https://github.com/your-name/my-project.git
   git branch -M main
   git push -u origin main
   ```
