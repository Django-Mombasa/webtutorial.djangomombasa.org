# Introduction

Welcome to the Django Mombasa Web Development Tutorial.

This tutorial walks you through the foundations you need before building
web applications with Django — from how the web works, to the languages
that power the front end, to the tools you will use every day.

## What you will learn

1. The Web and the Browser — how the internet and browsers work together.
2. HTML — structuring content on a page.
3. CSS — styling and laying out that content.
4. Basic JavaScript — adding interactivity in the browser.
5. Python — the programming language we use for Django.
6. Git and GitHub — tracking your code and collaborating with others.
