# Mazingira ya kipekee (virtual environments)

**Mazingira ya kipekee** huweka pakiti za Python za mradi mmoja mbali
na mfumo wako wote. Unda moja kwa kila mradi:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Mazingira yanapokuwa hai, `pip install` huweka pakiti ndani yake, si
kwenye mfumo mzima.
