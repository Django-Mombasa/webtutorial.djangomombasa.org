# If, for, na vijitendaji

```python
if umri >= 18:
    print("Mtu mzima")
else:
    print("Mtoto")

for tunda in matunda:
    print(tunda)

def salimu(jina):
    return f"Habari, {jina}!"

print(salimu("Dunia"))
```
