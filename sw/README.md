# Utangulizi

Karibu kwenye Mafunzo ya Utengenezaji wa Tovuti ya Django Mombasa.

Mafunzo haya yatakufundisha misingi unayohitaji kabla ya kujenga
tovuti kwa Django — kuanzia jinsi wavuti inavyofanya kazi, hadi lugha
zinazoendesha upande wa mtumiaji, hadi zana utakazozitumia kila siku.

## Utajifunza nini

1. Wavuti na Kivinjari — jinsi intaneti na vivinjari vinavyofanya kazi pamoja.
2. HTML — kupanga maudhui kwenye ukurasa.
3. CSS — kupamba na kupanga muonekano wa maudhui.
4. JavaScript ya Msingi — kuongeza mwingiliano kwenye kivinjari.
5. Python — lugha ya programu tunayotumia kwa Django.
6. Git na GitHub — kufuatilia msimbo wako na kushirikiana na wengine.

Tumia upau wa kando kuhama kati ya sura.
