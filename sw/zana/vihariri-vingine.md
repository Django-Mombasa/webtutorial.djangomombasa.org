# Vihariri vingine

Huhitaji kutumia IDE. Watengenezaji wengi wa Django wana raha na
kihariri chepesi pamoja na viongezi vichache.

## VS Code

[**Visual Studio Code**](https://code.visualstudio.com/) (VS Code)
ni kihariri cha bure kutoka Microsoft. Ni haraka, kinaweza
kubadilishwa sana, na kina viongezi kwa karibu kila kitu:

- Kiongezi cha **Python** kwa linting, utatuzi, na mazingira ya
  kipekee.
- **Pylance** kwa ukamilishaji unaotambua aina.
- Viongezi vya **Django** kwa kuangazia sintaksia ya templeti.
- **GitLens** kwa uunganishi bora wa Git.

## Cursor

[**Cursor**](https://www.cursor.com/) ni tawi la VS Code lenye
vipengele vya AI vilivyojengwa ndani ya kihariri — mazungumzo,
mabadiliko ya ndani, na upangaji upya wa aina ya mawakala ambao
wanaweza kutenda kwenye faili nyingi mara moja. Ikiwa unapenda VS
Code lakini unataka usaidizi wa AI kama kipengele kikuu, Cursor
kinaonekana cha kawaida kutoka siku ya kwanza.

## Kuchagua kimoja

- Ukitaka **zana kamili mara moja** → PyCharm.
- Ukitaka **kihariri chepesi kinachoweza kuongezwa** → VS Code.
- Ukitaka **AI kwanza** → Cursor.

Yoyote kati ya hivi itakupeleka kupitia mafunzo haya bila shida.
Chagua kile kinachohisi sawa na uendelee.
