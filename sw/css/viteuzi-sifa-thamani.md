# Viteuzi, sifa, na thamani

Sheria ya CSS ina sehemu tatu:

```css
kiteuzi {
  sifa: thamani;
}
```

- **Kiteuzi (selector)** — ni vipengele vipi vya kupamba (`h1`, `.kadi`, `#kuu`).
- **Sifa (property)** — ni nini cha kubadilisha (`color`, `margin`, `font-size`).
- **Thamani (value)** — mpangilio mpya (`red`, `10px`, `1.2em`).
