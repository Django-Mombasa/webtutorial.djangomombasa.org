# Classes na IDs

Ongeza sifa ya `class` au `id` kwenye HTML, kisha ilenge kwenye CSS:

```html
<p class="tahadhari">Kumbuka kuhifadhi kazi yako.</p>
```

```css
.tahadhari {
  background-color: #fff3cd;
  padding: 8px;
  border-left: 4px solid #f0ad4e;
}
```

Tumia **classes** pale vipengele vingi vinapokuwa na mitindo sawa.
Tumia **IDs** kwa vipengele vya kipekee tu.
