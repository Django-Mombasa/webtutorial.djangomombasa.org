# Git na GitHub

**Git** ni mfumo wa kudhibiti matoleo — hurekodi historia ya msimbo
wako ili uweze kurudi nyuma, kugawanya matawi, na kushirikiana kwa
usalama.

**GitHub** ni tovuti inayohifadhi hifadhi za Git mtandaoni, ili uweze
kushirikisha kazi yako na kufanya kazi na wengine.
