# Mtiririko wa msingi

```bash
git status               # ona kilichobadilika
git add .                # andaa mabadiliko
git commit -m "Ujumbe"   # rekodi picha ya wakati huo
```

**Commit** ni picha iliyohifadhiwa ya historia. Andika ujumbe mfupi na
ulio wazi unaoeleza *kwa nini* umefanya mabadiliko.
