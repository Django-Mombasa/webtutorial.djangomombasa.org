# Kufanya kazi na GitHub

1. Fungua akaunti kwenye [github.com](https://github.com/).
2. Tengeneza hifadhi mpya kwenye GitHub — **usiianzishe** na README
   ikiwa tayari una moja ndani ya kompyuta yako.
3. Unganisha mradi wa kompyuta yako na usukume:

   ```bash
   git remote add origin https://github.com/jina-lako/mradi-wangu.git
   git branch -M main
   git push -u origin main
   ```
