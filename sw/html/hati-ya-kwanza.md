# Hati yako ya kwanza ya HTML

```html
<!DOCTYPE html>
<html lang="sw">
  <head>
    <meta charset="UTF-8">
    <title>Ukurasa Wangu wa Kwanza</title>
  </head>
  <body>
    <h1>Habari, dunia!</h1>
    <p>Huu ni ukurasa wangu wa kwanza wa wavuti.</p>
  </body>
</html>
```

Ihifadhi kama `index.html` kisha ifungue kwenye kivinjari.
